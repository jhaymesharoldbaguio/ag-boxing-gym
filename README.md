# 🥊 AG Boxing Gym - Inventory Management System

Isang professional at real-time na inventory solution na binuo para sa **AG Boxing Gym**. Ang system na ito ay idinisenyo upang mapadali ang pagsubaybay sa mga sports equipment gamit ang modernong teknolohiya.

---

## 🌟 Key Features

- **🔐 Secure Access:** Nilagyan ng Firebase Authentication para masigurong si Coach lamang ang makakagamit ng system.
- **⚡ Real-time Updates:** Gamit ang Cloud Firestore, ang bawat dagdag o bawas sa stocks ay awtomatikong nag-uupdate sa lahat ng devices nang hindi kinakailangang i-refresh ang page.
- **📱 PWA (Progressive Web App):** Maaaring i-install sa Android at iOS device bilang isang "Mobile App" para sa mas mabilis na access.
- **🛠 Full CRUD Functionality:** Kakayahang mag-Add, Read, Update (Increment/Decrement), at Delete ng items sa inventory.
- **🎨 Modern Masculine Minimalist UI:** Disenyo na inspirado sa kulay ng ating bandila (Royal Blue, Crimson Red, at Golden Yellow) para sa matapang na "Pinoy Gym" vibe.

---

## 🚀 Tech Stack

- **Frontend:** HTML5, CSS3 (Custom Responsive Grid), JavaScript (ES6+)
- **Backend:** Firebase Authentication & Cloud Firestore
- **Deployment:** GitHub Pages
- **Development Tool:** Binuo nang buo sa isang **Android Mobile Device** gamit ang Acode at Keyboard/Mouse setup.

---

## 📲 Installation (PWA)

Para sa mas magandang karanasan, i-install ang app sa inyong home screen:

### For Android (Chrome):
1. Buksan ang website sa Google Chrome.
2. Hintayin ang "Install App" button sa itaas o pindutin ang **"Add to Home Screen"** sa browser menu.

### For iOS (Safari):
1. Buksan ang website sa Safari.
2. Pindutin ang **Share** button (square na may arrow pataas).
3. Piliin ang **"Add to Home Screen"**.

---

## 🛠 Project Structure

- `index.html` - Main structure at PWA configuration.
- `style.css` - Custom responsive styling (Mobile-First approach).
- `app.js` - Firebase logic (Auth, CRUD, at PWA install prompt).
- `manifest.json` - PWA app configuration at branding icons.
- `sw.js` - Service worker para sa offline support at installability.
